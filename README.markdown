# toilet

This is experimental software. Do not rely on it for strong and
secure ASCII art generation, it will only output ugly things.

See NEWS for the work in progress.

## Build requirements

 - libcaca >= 0.99.beta19
 - libtool
 - pkg-config
 - reasonably modern version of automake

## Build instructions

```
$ ./bootstrap
$ ./configure
$ make
$ sudo make install
```
